## مراحل راه اندازی 
( برای مشاهده تصاویر سایت قبل از راه اندازی به فولدر Picture در بالا مراجعه کنید )

بعد از باز کردن پروژه در IDE مورد استفاده (مـثلا VS Code)
### کدهای زیر را در ترمینال وارد کنید :
* **composer install**
* **npm install --force**
* **php artisan storage:link**
* **npm run dev**
* **php artisan serve**
